INTRODUCCION

Los sistemas de control de versiones, su importancia y sus características a lo largo de la historia.
Descripción general de lo que son Git y Github

Objetivo General


Comprender la importancia de los sistemas de control de versiones como GIT o GITHUB, para compartir y publicar archivos en formato HTML5 y CSS3 a la hora de desarrollar páginas WEB

Objetivos Especificos


*Conocer conceptos básicos sobre las herramientas de control de versiones
*Fortalecer la importancia de Git y GitHub como herramientas de control de versiones
*Aprender a manejar los clientes para escritorio de GIT y GitHub.
